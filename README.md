# POKEMON

Instalacion

```bash
npm install
```

Ejecutar

```
npm run dev
npm run serve
```

PARA XAMPP
Copiar en localhost
```
npm run build
```
ABRIR EN EL PUERTO http://localhost:5173/
Es necesario tener node js instalado https://nodejs.org/en
